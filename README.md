# Lab-9-ML

This Jupyter Notebook presents a machine learning project using a Random Forest model to predict whether a borrower will fully repay a loan. The dataset is obtained from LendingClub and includes loan records from 2007 to 2010.

The notebook begins with import the necessary Python libraries, including pandas, visualization tools, and scikit-learn.

Next, the notebook performs exploratory data analysis (EDA) by visualizing key variables (e.g., FICO score distributions) to better understand patterns in the data.

After that, the data is prepared for modeling by handling categorical variables (such as loan purpose) using dummy encoding. Finally, the notebook builds and trains a Random Forest classification model to predict loan repayment outcomes.
